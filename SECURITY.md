# Security Policy

## Supported Versions

We only provide security updates for the following versions:

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

The Pixielity team takes security seriously. If you find a vulnerability, please help us keep the community safe by following these steps:

1.  **Private Disclosure**: Send an email to security@pixielity.com with a detailed description of the vulnerability.
2.  **Acknowledgement**: We will acknowledge your report within 48 hours.
3.  **Fix and Release**: We aim to provide a fix within 14 days of confirmation.
4.  **Public Disclosure**: Once a fix is released, we will publicly announce the vulnerability and credit the reporter.

Please do not disclose the vulnerability publicly until we have had a chance to fix it.
