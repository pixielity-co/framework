# Pixielity Package B

Extended functionality module for the Pixielity Framework.

## Installation

```bash
composer require pixielity/package-b
```

## Usage

```php
<?php

use Pixielity\PackageB\Service;

$service = new Service();
```

## Testing

```bash
# Run package tests
composer test

# From root
vendor/bin/phpunit src/PackageB/tests
```
