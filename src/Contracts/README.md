# Pixielity Contracts

Framework interfaces and contracts for the Pixielity Framework.

## Installation

```bash
composer require pixielity/contracts
```

## Description

This package contains all the interfaces and contracts used throughout the Pixielity Framework. It provides abstractions for:

- Application contracts
- Container contracts
- Service provider contracts
- And more...

## Usage

```php
<?php

use Pixielity\Contracts\Application;
use Pixielity\Contracts\Container;

// Implement contracts in your classes
class MyApplication implements Application
{
    // Implementation
}
```
